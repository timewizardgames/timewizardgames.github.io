3D Overlay

This is my 3D livestream overlay that I use on my art streams!

It's built in Unity and uses OBS's Game Capture source to remove the transparent background. This allows perfect transparency without using color-keying, and even allows semi-transparency.

I made all of the models and textures for this project in Blender and Clip Studio Paint.

I also wrote a custom C# library to connect with Clip Studio Paint to get the current tool and color. This data affects the color and tool shown in the overlay!

Factory Factory

Factory Factory is a TV show about a chaotic, interactive, AI generated factory! It takes requests from Twitch chat to generate new content in realtime using ChatGPT and Unity.

This project was designed to run 24 / 7, but that proved too expensive, so now it streams live on Twitch 5 days a week!

About Me

Hi. I'm megalon.

I have 10+ years of programming experience, including 5 years using C# in Unity. In this time I created complex mods for 3 different Unity games: Beat Saber, Nickelodeon All-Star Brawl, and Ooblets. I have also released 2 small Unity projects of my own.

In 2018 I created the Beat Saber Modding Group, a Discord server with 200,000+ members dedicated to modding the VR game Beat Saber. I have collaborated on multiple programming projects through GitHub with other members of the community, including maintaining the Beat Saber mod manager Mod Assistant with 3 million+ downloads.

As you can see from this website, I have a lot of interests. Programming, 3D modeling, digital painting, and music production are all things that I have a passion for. I believe all of these contribute to my skills in Game Development.
